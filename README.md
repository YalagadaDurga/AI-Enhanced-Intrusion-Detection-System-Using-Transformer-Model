# AI-Enhanced Intrusion Detection System using Transformer Model

## 📌 Project Overview
This project presents an AI-based Intrusion Detection System (IDS) that uses a Transformer-based deep learning model to detect malicious network traffic. The system analyzes network packets and classifies them as either normal or attack traffic.

The project aims to improve cybersecurity by identifying various types of network intrusions such as DoS, Probe, R2L, and U2R attacks.

---

## 🎯 Objectives
- Detect network intrusions using machine learning
- Improve accuracy using Transformer architecture
- Provide real-time intrusion detection
- Visualize results using an interactive dashboard

---

## 🧠 Technologies Used
- Python
- PyTorch
- Scikit-learn
- Pandas & NumPy

- Matplotlib & Seaborn

---

## 📊 Dataset
The model is trained using the **NSL-KDD dataset**, a benchmark dataset for intrusion detection research.

Dataset includes various attack categories:
- DoS
- Probe
- R2L
- U2R
- Normal Traffic

Note: Dataset is not included in the repository due to size limitations.
